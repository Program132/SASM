# MOV Instruction

**MOV** (means *MOVE*) is SASM, it's to give a value to a possible register.

A register is like a variable in current langages programming, but it's limited and there is only 13 registers (R0 to R12).

## Format
``MOV Re, #Value``

## Example
``MOV R5, #10``

``MOV R0, #40``

``MOV R12, #150``
